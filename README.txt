AluraFrame:
Um site de negociações.
Feito com HTML, CSS e JS Avançado.

Alura-typer:
Um jogo de digitação.
Temos uma página inicial que mosta como é o jogo e algumas 
de suas funções. Usamos templates nessa página para fazer 
a funcionalidade de passar as telas lado a lado.
(completar)
feito com HTML, CSS, JS, jQuery e Ajax.

ALurinha:
Um site de cursos que apresenta opções de pagamentos e mostra
os cursos disponíveis. O intuito foi treinar habilidades de
flexbox. 
Feito com HTML e CSS.

Barbearia:
Um site de uma barbearia com opçoes de cadastro, mostra os 
serviços oferecidos e uma apresentação da barbearia.
Feito com HTML e CSS.

Nutricionista:
Um site feito para uma nutricionista que contém o peso a 
altura e a gordura corporal do paciente e calcula automaticamente
o imc. Tem um filtro para encontrar os pacientes, um Buscar 
Pacientes que pega pacientes de outro servidor e adiciona na
tabela atual e podemos adicionar novos pacientes. E clicando
2x na tabela em uma linha essa linha será excluida.
Feito com HTML, CSS, jQuery, DOM, AJAX.

Portal News:
Um portal de notícias, o principal objeto com essa página foi 
terinar as habilidades do Design Responsivo do CSS3, criando uma
tela adaptável de acorso com o tamanho da janela do navegador do
usuário.
Feito com HTML, CSS e JS.

Telefonia:
Uma tabela dinâmica da qual podemos adicionar novos modelos de 
celualar em conjunto com seu preço e outras especificações.
Feito com HTML, CSS e JS.

WWC:
Um site de apresentação persoal feito em conjunto com o grupo
Women Who Code Recife. 
Feito com HTML e CSS.
